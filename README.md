# 基于LLM的占星骰子占卜插件
## 占星骰子占卜
用法: /占星 [你的问题]
示例: /占星 我该怎样改善感情关系?

## 查询占星符号含义
用法: /symbol [符号名称]
示例: /symbol 天蝎座
